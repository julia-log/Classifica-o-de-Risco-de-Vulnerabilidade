# Classifica-o-de-Risco-de-Vulnerabilidade
Durante as aulas de Análise e Desenvolvimento de Sistemas (ADS), foi passada uma lista de exercícios práticos com o objetivo de aprimorar o raciocínio lógico e a aplicação da linguagem Python em situações reais. Nos exercícios I1 e I2, pude desenvolver e aplicar meus conhecimentos sobre estruturas condicionais, operadores lógicos e manipulação de dicionários, simulando cenários comuns em ambientes de T.I e Segurança da Informação, como classificação de riscos e controle de usuários em sistemas.

I1 – Classificação de Risco de Vulnerabilidade:
Criei um script que classifica o nível de risco com base em uma pontuação (score CVSS) e fornece uma recomendação de ação adequada.

I2 – Gerenciamento de Inventário de Usuários:
Desenvolvi um programa que utiliza dicionários para representar e atualizar dados de usuários, verificando chaves, adicionando informações e modificando valores.

Essas atividades foram fundamentais para consolidar conceitos de lógica de programação, controle de fluxo e estruturas de dados em Python.
